# Boardgame Shelf

A PWA for managing your board game collection, logging play sessions, and tracking stats — all offline, no backend.

## Features

- **Collection** — catalog your board games with details (players, time, complexity, category)
- **Play Sessions** — log plays with players, scores, winners, and notes
- **Stats** — visualize your gaming habits, most played games, win rates
- **Settings** — theme, accent color, data export/import

## Tech Stack

- React 19 + TypeScript
- Vite + Tailwind CSS 4 (glass morphism UI)
- Zustand (state persisted to IndexedDB)
- React Router 7
- Framer Motion
- Radix UI primitives
- PWA (offline-first via Workbox)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
